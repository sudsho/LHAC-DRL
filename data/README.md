# Data

The original industrial production data used to calibrate the synthetic
generators in `lhac/data.py` is provided by an industry partner under
confidentiality restrictions and is not redistributed in this
repository.

The synthetic generator in `lhac.data.DataGenerator` reproduces the
three arrival distributions, the three two-test-cell ratios, and the
processing-time / due-date statistics used in the paper. It is fully
deterministic when seeded.

```python
from lhac.data import DataGenerator
df = DataGenerator(pattern='uniform', n_servers=300,
                   twotc_ratio=0.20, horizon_days=10, seed=0).as_dataframe()
df.to_csv('data/example.csv', index=False)
```

The benchmark grid (3 patterns × 3 sizes × 3 2TC ratios = 27 instances)
can be regenerated by iterating `lhac.data.benchmark_grid()`.
